# AI Email Spam Detection System

##  Overview
This project detects whether a message is Spam or Not Spam using Machine Learning and NLP.

##  Features
- Text preprocessing (lowercase, punctuation removal, stopwords removal)
- TF-IDF vectorization
- Logistic Regression model (~95% accuracy)
- Flask web application for real-time prediction

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Flask

## How to Run
1. Install dependencies:
   pip install flask sklearn nltk pandas

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000/
